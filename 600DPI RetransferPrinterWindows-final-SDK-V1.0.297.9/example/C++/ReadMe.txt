
转印机卡片打印机标准库示例程序（控制台）

=====================================
各示例程序介绍

ContactCardConsole：
用于带内置接触式德卡读卡器T10的卡片机，可读带外置芯片的接触式CPU卡

ContactlessConsole：
用于带内置非接触式德卡读卡器T10的卡片机，可读带内置芯片的非接触式CPU卡

DeCard3In1Console：
用于带内置德卡三合一（T10）卡片机

DrawPrintConsole:
专用于套打

QueryStatusConsole：
可用于查询状态的示例程序

=====================================

使用方法：
1、打开想要演示的示例程序文件夹；
2、双击sln文件（需安装VS2015）；
3、生成项目；
4、运行；
5、根据控制台提示执行想要的效果；
